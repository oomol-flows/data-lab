# data-lab

[English](./README.md) | 中文

## 简介

data-lab 是一个用于数据分析和图表绘制的 OOMOL block 库，底层引用了 pandas 和 plotly 两个著名的 Python 库。

## 功能
- 数据分析：利用 pandas 提供强大的数据处理功能。
- 图表绘制：利用 plotly 提供交互式、高质量的数据可视化。
- 简单易用：通过 OOMOL block 的封装，简化了复杂的数据分析与可视化流程。
